# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2026-07-05

### Added

- `V1`, `V3`, `V4`, `V5`, `V6`, `V7` generators returning base62-encoded SUUIDs.
- `Encode` and `Decode` for converting standard UUIDs to and from SUUIDs.
- Built on `github.com/gofrs/uuid`, which is MIT-licensed and supports UUID v1 through v7.
